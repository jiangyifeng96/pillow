## Chinese Output Rules

Use direct, natural Chinese.

- Avoid report-sounding phrases like `综上所述`, `整体而言`, `值得注意的是`.
- Avoid empty transitions like `基于此`, `由此可见`, `进一步来说`.
- Prefer concrete statements over summary slogans.
- When listing findings, state the problem first, then the consequence.
- For GitHub comments, keep the tone like a teammate, not a generated report.
